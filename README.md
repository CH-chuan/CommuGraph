CommuGraph is a tool designed to **visualize and analyze multi-agent chat logs**, providing deep insights into communication patterns, agent roles, and emergent structures. While many existing approaches rely on basic metrics like message counts, CommuGraph uncovers the **structure and flow** of conversations, helping researchers and developers make sense of complex multi-agent dynamics.

## ğŸ” Overview

Multi-agent systems often operate through long-form chats that obscure key interaction dynamics. CommuGraph tackles this by transforming chat logs into **dynamic communication graphs**, where:

- **Nodes** represent agents, with size indicating activity level.
- **Edges** represent message flow, with thickness reflecting frequency and color indicating dominant intent (e.g., delegation, inquiry).

These graphs are then analyzed using classic network algorithms (e.g., centrality, clustering) to surface dominant, isolated, or emergent patterns in agent communication.
